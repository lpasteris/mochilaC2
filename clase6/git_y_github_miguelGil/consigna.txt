a. ¿Quién inventó el sistema de control de versión Git y por qué?
Fue desarrollado originalmente por Linus Torvalds, pensando en la eficiencia, la confiabiliadad y la compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código de fuente.
