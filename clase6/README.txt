a. ¿Quién inventó el sistema de control de versión Git y por qué?
Linus Torvalds, el creador de Linux. Se baso en BitKeeper y fue diseñado como un sistema de control de versiones de bajo nivel con ciertos objetivis basados en la velocidad, diseño sencillo, apoyo en el desarrollo no lineal (ramas), distribuido, manejo de grandes proyectos.

b. ¿A quién pertenece actualmente Github y por qué?
En 2018 Microsoft compró GitHub y su mantenimiento está supervisado por Junio Hamano (2009)
