a. ¿Quién inventó el sistema de control de versión Git y por qué?
Git es un proyecto de código abierto maduro y con un mantenimiento activo que desarrolló originalmente Linus Torvalds
pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente.
b. ¿A quién pertenece actualmente Github y por qué?
Desde el año 2007 pertenece a Microsoft, que con el control de GitHub, se volvió la comunidad de desarrollo más nutrida del planeta.
c. ¿Hay otra forma que no sea la terminal para trabajar con Github?
Se puede trabajar directo desde la plataforma web y desde vsCode.
