a. ¿Quién inventó el sistema de control de versión Git y por qué?
Git es un proyecto de código abierto maduro y con un mantenimiento activo que desarrolló originalmente Linus Torval s
pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente. Este sistema fue creado para reemplazar el sistema anterior dado que este ultimo dse transformo a de pago, mientras que git se mantuvo en modo source.
