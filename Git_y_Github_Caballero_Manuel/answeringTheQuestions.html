Answering The Questions


1) ¿Quién inventó el sistema de control de Git y por qué?

R: Como muchas de las grandes cosas en esta vida, Git comenzó con un poco de destrucción creativa y una gran polémica. Es un software de control de versiones diseñado por Linus Torvalds pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente. Se creo en base a que anteriormente Linux contaba con un software que se llamaba "Bitkeeper" que en un minuto dejo de ofrecerce de manera gratuita, dando paso a la obligacion de contar con un software libre y robusto.


2) ¿A quién pertenece actualmente Github y por qué?

R: En el 2018 Microsoft anuncío y compró respectivamente la compañia por un importe de más de 7000 millones de dólares. Desde el cambio de CEO para Microsoft la compañia ha adoptado una estrategia nueva, mucho más abierta al mundo del software libre y a la integración con otras plataformas y servicios.


3) ¿Hay otra forma que no sea la terminal para trabajar con Github?

R: Si, existe la aplicacion Github Desktop que te permite trabajar con Github sin necesidad de utilizar la terminal.
