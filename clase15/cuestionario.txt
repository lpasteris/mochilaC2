¿Qué tipo de máquina virtual es virtualBox?.
R: Es un software de virtualización para arquitecturas x86/amd64

demás? ¿por qué?
R:Una máquina virtual es un software que permite emular el funcionamiento de un ordenador dentro de otro ordenador, gracias a un proceso de encapsulamiento que aísla a ambos, por lo cual las demás no se verán afectadas
